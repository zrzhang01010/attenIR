# AttenIR: Aligning dense visual attention with mesh warping for unsupervised image retargeting

## Dataset

We use the ObjectIR dataset to train and evaluate our method. Please refer to [ObjectIR](https://github.com/tlliao/Object-IR) for more details about this dataset. The other test dataset refers to RetargetMe, please refer to [RetargetMe](https://people.csail.mit.edu/mrub/retargetme/).
The dataset is also available for download [Baidu Cloud](https://pan.baidu.com/s/1Wbz4U_2otUHAUoMlXn1Obg?pwd=z58r)(Extraction code:z58r)
